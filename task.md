## هيكل المشروع

```
movie_bot/
├── main.py              # نقطة الدخول
├── config.py            # الإعدادات
├── bot.py               # معالجات البوت
├── search.py            # البحث عن الأفلام
├── downloader.py        # تحميل التورنت
├── splitter.py          # تقسيم الفيديو
├── requirements.txt     # المكتبات
├── downloads/           # مجلد التحميلات المؤقتة
└── task.md              # خطة المشروع
```

## التدفق

```
1. المستخدم ← يكتب اسم فيلم
2. البوت ← يبحث في YTS ويعرض قائمة
3. المستخدم ← يختار الفيلم
4. البوت ← يعرض الجودات المتاحة
5. المستخدم ← يختار الجودة
6. البوت ← يحمل التورنت
7. البوت ← يرسل الفيديو (مقسماً إذا كان > 3GB)
8. البوت ← يحذف الملفات المؤقتة
```
